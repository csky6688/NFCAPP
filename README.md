# NFCAPP

利用网络框架OkHttp、Litepal数据库框架以及手机里面的NFC功能开发的巡检系统,
配合我前面提交的NFCDemo读写卡的APP,首先给巡更卡进行编号,然后利用NFCAPP读取编号
同时提交给后台,后台返回编号对应的真实场景的位置描述展示给用户
